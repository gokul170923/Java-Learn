JAVA.base module:
        java.lang package :
                Object class : 
                              .hashCode() , gives a integer from its reference address that can be used in hashable containers.
                              .equals() , in Object default wraps == ( this == obj) but can be overridden for values.
                              .getCLass() , returns the runtime class of the object no matter the type of the refrence variable.
                String class : 
                              immutable
                              string literal like "java" are put in SCP so tehy have same refrence
                              .equals() is overriden to check the actual value of string hence == is not same as .equals()
                              new String("java") forces the jvm to make a new string object that is not kept in SCP
                              .length()	, Number of characters
                              .charAt()	, Character at index
                              .substring(begin) or (beginIn,endEX) , 	Creates new string
                              .equalsIgnoreCase()	, Case-insensitive compare
                              .contains()	 , Substring check
                              .split("delemeter")	, Regex-based split retuns a String[]
                              .replace('a','b') or ("before","after")	 , Returns new string
                              .toCharArray() converts string to a char[] array
                              .intern() , if the string is present in SCP returns the refrnece else creates one in the SCP
                StringBuilder : 
                               muttable
                               StringBuilder() , StringBuilder(CharSequence seq) , StringBuilder(int capacity) , StringBuilder(String str)
                               .append() , This method appends the given string argument to the sequence.
                               .capacity() , gives current capacity of the container , increases by  c * 2  + 2
